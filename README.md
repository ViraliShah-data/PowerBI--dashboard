# 📊 Sales Pipeline Dashboard (Power BI)

## Overview

This is one of my first dashboards created while learning Power BI. The project focuses on analyzing a sales pipeline to understand how leads move across different stages and how it impacts overall revenue.
The aim was to track key metrics and identify areas where the sales process can be improved.

## What I Built

* Designed an interactive dashboard to monitor pipeline stages from lead generation to deal closing
* Included key metrics such as total revenue, average deal size, and lead conversion rate
* Added filters to analyze performance by product category,country and year

## Key Insights

* Observed a drop in conversions between negotiation and closing stages
* Revenue trends showed variation across different months
* Certain regions contributed more consistently to overall revenue
* Conversion rate suggests scope for improving pipeline efficiency
